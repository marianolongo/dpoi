El problema con el tag br

Es incorrecto el uso del tag br ya que todo lo relacionado con el estilo de una pagina deberia estar contemplado en el css. El HTML solamente deberia tener la estructura de la pagina.

Definiciones

  • URI

    Cadena de caracteres que identifica un recurso abstracto o físico

  • URL

    Uniform Resource Locator. La dirección específica que se asigna a cada uno de los recursos disponibles en la red (Para que puedan ser localizados o identificados)

  • HTML

    Hyper Text Markkup Language. Se encarga de desarrollar una descripción sobre los contenidos que aparecen como textos y sobre su estructura, complementando dicho texto con diversos objetos

  • XHTML

    eXtensible HyperText Markup Language. HTML expresado como XML valido. Es mas estricto a nivel tecnico. Por ejemplo, si o si hay que cerrar los tags.

  • CSS

    Cascading Style Sheets. Sirve para determinar la manera en la que se va a presentar el contenido de distintos lenguajes de markup (Por ejemplo HTML)
